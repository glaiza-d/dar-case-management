Tech Stack:
Django
React
Postgres or MySQL

Core Features:

- Authentication and authorization (JWT and RBAC) 
- User management (add users, roles, and permissions)
- Case management (create, update, assign, search, and view records) 
- Case workflow automation (case assignment rules, status change notifications, prioritization)
- File and document storage (import data and upload documents) 
- Case prioritization, classification, and analytics dashboard

User Roles:

Admin User - Full access
Editor User - Create, update, search, and view records
Viewer User - Search and view records
